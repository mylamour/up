# Project Target (North Star)

**Updated**: 2026-02-27  
**Purpose**: Single reference for where the project is heading. Use this to align work and measure progress.

---

## Vision (Final Target)

> **Make AI-assisted development verifiable, observable, and safe — turning vibe coding into software engineering.**

- **Verifiable**: Every change testable, traced, and validated.
- **Observable**: Full visibility into state, provenance, and metrics.
- **Safe**: Checkpoints, circuit breakers, doom loop detection, context budget.
